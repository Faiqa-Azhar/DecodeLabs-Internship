# Mall Customer Dataset Analysis & Machine Learning Project

## Project Overview

This project performs complete data analysis, preprocessing, visualization, and machine learning on the Mall Customer Dataset.

The main goal of this project is to understand customer behavior, clean and preprocess the data, analyze spending patterns, and build machine learning models for prediction and classification.


# Dataset Information

The dataset contains customer information collected from a shopping mall.

## Features in Dataset

| Column Name | Description |
|---|---|
| CustomerID | Unique customer ID |
| Gender | Male or Female |
| Age | Customer age |
| Annual Income (k$) | Annual income of customer |
| Spending Score (1-100) | Spending score assigned by mall |

# Objectives

- Understand dataset structure
- Perform exploratory data analysis (EDA)
- Handle missing values
- Remove duplicate data
- Encode categorical variables
- Detect and remove outliers/noise
- Perform feature engineering
- Visualize customer behavior
- Apply regression and classification models
- Generate business insights and recommendations

# Key Insights

- Customers with higher income often show higher spending behavior.
- Spending patterns vary across age groups.
- Data preprocessing improved model quality.
- Outliers and noise were successfully removed.
- Machine learning models predicted customer behavior effectively.


# Business Recommendations

- Target high-spending customers with premium offers.
- Use age-based marketing strategies.
- Improve engagement for low-spending customers.
- Focus marketing campaigns on customers with higher annual income.
